每天一个linux命令(05)：rm

发表于 2018-04-13 | 分类于 linux系统

既然可以利用mkdir创建目录，那当然有一个命令可以来删除它，那就是rm。

rm命令是remove的缩写，也是比较常用的命令，它的功能是删除一个目录中的一个或多个文件或目录，它也可以将某个目录及其下的所有文件及子目录均删除。

当然这需要在命令后跟一些命令参数，比如说-rf，对与这个命令一直有一个笑话，就是有人在网上问不小心执行了rm -rf /后怎么办，得票最高的答案是凉办。所以在执行rm之前最好先确认一下在哪个目录，到底要删除什么东西，操作时保持高度清醒的头脑。另外对于链接文件，只是删除了链接，原有文件均保持不变。

rm

常用参数

参数 描述
-f –force 忽略不存在的文件，从不给出提示。
-i –interactive 进行交互式删除
-r –recursive 指示rm将参数中列出的全部目录和子目录均递归地删除。
-v –verbose 详细显示进行的步骤

用法

rm

不带命令参数一般会有提示

-f 强行删除文件file，系统也不再提示

-i 逐一提示删除文件file

-r 递归删除目录及子目录中所有档案删除file

-rf 如果文件太多，用r参数就会很麻烦。这时可以用不提示地强行递归删除目录及子目录中所有档案删除file，这是一个命令参数组合，当然前提是删除的目标是百分百确定要删除的。

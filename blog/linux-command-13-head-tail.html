每天一个linux命令(13)：head - tail

发表于 2018-04-24 | 分类于 linux系统

head 与 tail 就像它的名字一样的浅显易懂，它是用来显示开头或结尾某个数量的文字区块，head 用来显示档案的开头至标准输出中，而 tail 想当然尔就是看档案的结尾。

less

常用参数

参数 描述
-c <数目> 显示的字节数
-n <行数> 显示行数
- q head 隐藏文件名, tail 不显示处理信息
-f tail 循环读取
-s tail –sleep-interval=S 与-f合用,表示在每次反复的间隔休眠S秒
